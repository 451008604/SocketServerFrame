# NETS 简介

一个追求轻量、性能、实用、可快速上手的网络框架。采用工作池模式，已实现协程复用并且可根据并发数量自动扩容协程池。连接建立只需占用3个协程（1个读协程、1个写协程、1个协程池内的工作协程）。

使用面向接口编程和组合设计模式，最大程度提高系统的灵活性、可维护性和可扩展性

**拒绝炫技，保证代码首先要通俗易懂，其次再保证机器运行正常**

现已支持：

* 服务：
  - TCP
  - WebSocket(s)
* 协议：
  - ProtocolBuffer
  - JSON
* 功能：
  - 设置连接建立时的前置
  - 设置连接断开时的后置
  - 绑定消息属性
  - 消息处理中间件
  - 自定义编码/解码器
  - 频道广播
  - 全服广播
  - 消息业务panic阻断
  - 停服时优雅关闭所有连接
